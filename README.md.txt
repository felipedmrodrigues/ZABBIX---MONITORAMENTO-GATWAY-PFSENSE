# ZABBIX - Monitoramento de Gateway pfSense

Esse projeto contém:

- Script PHP para retornar status dos gateways do pfSense para o Zabbix.
- Exemplo de configuração do `zabbix_agentd.conf`.

## Arquivos

- `gateway.php`: Script para discovery e status dos gateways.
- `zabbix_agentd.conf.exemplo`: Exemplo de configuração para incluir no agente do Zabbix.
